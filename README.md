Credit Card Fraud Detection using Machine Learning.  Implemented Random Forest with SMOTE to handle class imbalance and accurately detect fraudulent transactions.

Credit Card Fraud Detection using Machine Learning. This project focuses on detecting fraudulent credit card transactions using machine learning techniques.

___________________________________________________________________________________________________________________________________________________

🔍 Overview

The dataset contains transaction details labeled as:

0 → Normal Transaction  
1 → Fraudulent Transaction  

Since the dataset is highly imbalanced, SMOTE (Synthetic Minority Over-sampling Technique) was applied to balance the data.

___________________________________________________________________________________________________________________________________________________

⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Imbalanced-learn
- Google Colab

____________________________________________________________________________________________________________________________________________________

🚀 Methodology

1. Data preprocessing and cleaning  
2. Removal of non-numeric features  
3. Feature scaling using StandardScaler  
4. Handling class imbalance using SMOTE  
5. Model training using Random Forest Classifier  
6. Prediction of transactions (Fraud/Normal)  
7. Model evaluation using:
   - Confusion Matrix  
   - Accuracy Score  
   - Classification Report

____________________________________________________________________________________________________________________________________________________

📊 Results

The model successfully detects fraudulent transactions with an accuracy of around 90%+.
